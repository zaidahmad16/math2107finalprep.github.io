4.1

Subspaces of a Vector Space

HW 1–2 · Quiz 1 · Midterm Q4

Definitions to know cold

MEMORIZE

Vector Space. A set $V$ with addition and scalar multiplication satisfying 10 axioms. Main examples: $\mathbb{R}^n$, $\mathcal{P}_n$, $M_{m\times n}$, $C(\mathbb{R})$.

MEMORIZE

Subspace. $W\subseteq V$ is a subspace if: (i) $\mathbf{0}\in W$, (ii) $\mathbf{u},\mathbf{v}\in W\Rightarrow\mathbf{u}+\mathbf{v}\in W$, (iii) $\mathbf{v}\in W, c\in\mathbb{R}\Rightarrow c\mathbf{v}\in W$.

Key facts

Dimension Table
SpaceZero vectordimStandard basis
$\mathbb{R}^n$$(0,\ldots,0)$$n$$\{e_1,\ldots,e_n\}$
$\mathcal{P}_n$$p(x)=0$$n+1$$\{1,x,x^2,\ldots,x^n\}$
$M_{m\times n}$zero matrix$mn$$\{E_{ij}\}$
⚠ Common Traps

$\dim(\mathcal{P}_3)=4$, not 3. Basis is $\{1,x,x^2,x^3\}$ — four vectors. This is Mezo's most-tested trap.

$\mathbb{R}^2$ is NOT a subspace of $\mathbb{R}^3$ — different sets (2-entry vs 3-entry vectors).

Union of subspaces is usually NOT a subspace. Intersection always is.

Proof recipe — Subspace (6 marks on midterm)

3-Step Subspace Proof Template
1Zero (2 pts): Identify the zero of $V$ explicitly. Verify it satisfies $W$'s condition. Conclude $\mathbf{0}\in W$.
2Addition (2 pts): "Let $f,g\in W$" (assume both satisfy the condition). Compute $f+g$. Show it satisfies the condition. Conclude $f+g\in W$.
3Scalar (2 pts): "Let $f\in W$ and $c\in\mathbb{R}$." Compute $cf$. Show it satisfies the condition. Conclude $cf\in W$. $\square$

Worked Example

Midterm Q46 marks

Show $W=\{f\in V:f(1)=f(0)\}$ is a subspace of $V$.

Solution
1Zero: $z(x)=0$ satisfies $z(1)=0=z(0)$. So $z\in W$. ✓
2Addition: Let $f,g\in W$. $(f+g)(1)=f(1)+g(1)=f(0)+g(0)=(f+g)(0)$. So $f+g\in W$. ✓
3Scalar: Let $f\in W$, $c\in\mathbb{R}$. $(cf)(1)=cf(1)=cf(0)=(cf)(0)$. So $cf\in W$. $\square$ ✓

True / False Practice

1. $\mathcal{P}_2$ is a subspace of $\mathcal{P}_3$.
True (Quiz 1)

Every polynomial of degree $\leq 2$ also has degree $\leq 3$, so $\mathcal{P}_2\subseteq\mathcal{P}_3$. The subspace conditions are inherited.

2. $H=\{p\in\mathcal{P}_2:p(x)=a+x^2,\,a\in\mathbb{R}\}$ is a subspace of $\mathcal{P}_2$.
False (Quiz 1)

Take $q=1+x^2\in H$. Then $2q=2+2x^2\notin H$. Fails scalar closure.

3. The set of all invertible $2\times2$ matrices is a subspace of $M_{2\times2}$.
False

The zero matrix is not invertible, so condition (i) fails.

4. The union of two subspaces of $V$ is always a subspace.
False

$H_1=\text{Span}\{(1,0)\}$, $H_2=\text{Span}\{(0,1)\}$: $(1,0)+(0,1)=(1,1)\notin H_1\cup H_2$.

4.2

Null Spaces, Column Spaces & Linear Transformations

HW 1–2 · Midterm Q7
MEMORIZE

Linear Transformation. $T:V\to W$ is linear if for all $\mathbf{v},\mathbf{u}\in V$ and $c\in\mathbb{R}$: (1) $T(\mathbf{v}+\mathbf{u})=T(\mathbf{v})+T(\mathbf{u})$   (2) $T(c\mathbf{v})=cT(\mathbf{v})$

MEMORIZE

Null Space: $\text{Nul}(A)=\{\mathbf{x}\in\mathbb{R}^n:A\mathbf{x}=\mathbf{0}\}$ — subspace of $\mathbb{R}^n$.

Column Space: $\text{Col}(A)=\text{Span}\{\mathbf{a}_1,\ldots,\mathbf{a}_n\}$ — subspace of $\mathbb{R}^m$.

Key Facts

Rank-Nullity: $\dim(\text{Nul}(A))+\dim(\text{Col}(A))=n$.

Col($A$) basis: Use pivot columns of the original $A$ (not REF!).

$T$ is one-to-one $\Leftrightarrow$ $\ker(T)=\{\mathbf{0}\}$.

⚠ Don't mix these up

If $A$ is $3\times5$: Nul lives in $\mathbb{R}^5$, Col lives in $\mathbb{R}^3$.

True / False Practice

5. $T(x,y)=(|x|,y)$ is a linear transformation $\mathbb{R}^2\to\mathbb{R}^2$.
False

$T(-1\cdot(1,0))=T(-1,0)=(1,0)$ but $-1\cdot T(1,0)=(-1,0)$. Fails homogeneity.

6. If $A$ is $3\times5$, then $\dim(\text{Nul}(A))\geq2$.
True

At most 3 pivots, so at least $5-3=2$ free variables.

4.3–4.5

Linear Independence, Bases & Dimension

Quiz 2 · Midterm Q1, Q2, Q9
MEMORIZE — Midterm Q1

Linear Independence. $\{\mathbf{v}_1,\ldots,\mathbf{v}_k\}\subseteq V$ is linearly independent if $c_1\mathbf{v}_1+\cdots+c_k\mathbf{v}_k=\mathbf{0}$ implies $c_1=\cdots=c_k=0$.

MEMORIZE

Basis. A set $\mathcal{B}$ is a basis for $V$ if it is linearly independent AND spans $V$.

Dimension. $\dim(V)=$ number of vectors in any basis for $V$.

Theorems 10–13 (Mezo's notation)

Let $\dim(V)=n$. Then: any set of $>n$ vectors in $V$ is linearly dependent. Any linearly independent set of exactly $n$ vectors is a basis. Any spanning set of exactly $n$ vectors is a basis. Any spanning set contains a basis. Any linearly independent set can be extended to a basis.

Coordinate Vectors (§4.4)

If $\mathcal{B}=\{\mathbf{b}_1,\ldots,\mathbf{b}_n\}$ is a basis and $\mathbf{v}=c_1\mathbf{b}_1+\cdots+c_n\mathbf{b}_n$, then $[\mathbf{v}]_\mathcal{B}=(c_1,\ldots,c_n)^\top\in\mathbb{R}^n$.

Proof recipe — Linear Independence (6 marks on midterm)

Testing Linear Independence in Any Space
1Set up: "Suppose $c_1\mathbf{v}_1+c_2\mathbf{v}_2+c_3\mathbf{v}_3=\mathbf{0}$ for $c_1,c_2,c_3\in\mathbb{R}$."
2Expand and collect: Substitute the actual vectors. Group by type.
3Form system: Each component gives one equation.
4Row reduce: Show each step clearly.
5Conclude: "Only solution is $c_1=c_2=c_3=0$. Therefore linearly independent."

Worked Example

Midterm Q2 — exact6 marks

Is $\{p_1=x^2+x+1,\,p_2=x^2+1,\,p_3=x+1\}$ linearly independent in $\mathcal{P}$?

Solution
1$c_1(x^2+x+1)+c_2(x^2+1)+c_3(x+1)=0$
2$(c_1+c_2)x^2+(c_1+c_3)x+(c_1+c_2+c_3)=0$
3System: $c_1+c_2=0$, $c_1+c_3=0$, $c_1+c_2+c_3=0$
4Row reduce to $I_3$. Unique solution: $c_1=c_2=c_3=0$.
5Therefore $\{p_1,p_2,p_3\}$ is linearly independent. $\square$

True / False Practice

7. If $V=\text{Span}\{\mathbf{v}_1,\ldots,\mathbf{v}_k\}$ then some subset is a basis for $V$. (Midterm Q9a)
True

Remove any vector that is a linear combination of the rest. Repeat. The remaining set spans $V$ and is linearly independent.

8. $\mathbb{R}^3$ and $\mathcal{P}_3$ are isomorphic. (Midterm Q9b)
False

$\dim(\mathbb{R}^3)=3$ but $\dim(\mathcal{P}_3)=4$. Isomorphic spaces must have equal dimension.

9. If $\dim(V)=n$, there exists a spanning set for $V$ with $n+1$ vectors. (Midterm Q9c)
True

Take any basis and add $\mathbf{v}_{n+1}=\mathbf{0}$. The span is unchanged since $\mathbf{0}$ contributes nothing.

4.6

Change of Basis & Matrix Representations

HW 2–3 · Quiz 2–3 · Midterm Q6, Q7
MEMORIZE — Midterm Q6

Change-of-Coordinates Matrix. If $\mathcal{A}=\{\mathbf{a}_1,\mathbf{a}_2,\mathbf{a}_3\}$ and $\mathcal{B}=\{\mathbf{b}_1,\mathbf{b}_2,\mathbf{b}_3\}$ are bases for $V$:

$$P_{\mathcal{B}\leftarrow\mathcal{A}}=\bigl[[\mathbf{a}_1]_\mathcal{B}\;\;[\mathbf{a}_2]_\mathcal{B}\;\;[\mathbf{a}_3]_\mathcal{B}\bigr] \quad\text{and}\quad [\mathbf{v}]_\mathcal{B}=P_{\mathcal{B}\leftarrow\mathcal{A}}[\mathbf{v}]_\mathcal{A}$$
Similar Matrices (§4.6 / §5.3)

$A$ and $B$ are similar if $A=PBP^{-1}$ for some invertible $P$. Similar matrices have the same: eigenvalues, characteristic polynomial, determinant, trace, rank.

Worked Example — Change of Basis

Midterm Q6 — exact6 marks

$\mathbf{a}_1=2\mathbf{b}_1-\mathbf{b}_2$, $\mathbf{a}_2=-\mathbf{b}_1+\mathbf{b}_2+\mathbf{b}_3$, $\mathbf{a}_3=\mathbf{b}_2-2\mathbf{b}_3$. Find $P_{\mathcal{B}\leftarrow\mathcal{A}}$ and $[\mathbf{v}]_\mathcal{B}$ for $\mathbf{v}=\mathbf{a}_1+\mathbf{a}_2-\mathbf{a}_3$.

Solution
aRead off coefficients: $P_{\mathcal{B}\leftarrow\mathcal{A}}=\begin{pmatrix}2&-1&0\\-1&1&1\\0&1&-2\end{pmatrix}$.
b$[\mathbf{v}]_\mathcal{A}=(1,1,-1)^\top$. Then $[\mathbf{v}]_\mathcal{B}=P_{\mathcal{B}\leftarrow\mathcal{A}}(1,1,-1)^\top=(1,-1,3)^\top$.
5.1–5.3

Eigenvalues & Eigenvectors

Quiz 3–4 · Midterm Q5, Q8
MEMORIZE — Midterm Q5

Eigenvalue / Eigenvector. $\lambda\in\mathbb{R}$ is an eigenvalue of $A\in M_{n\times n}$ if there exists a nonzero $\mathbf{v}\in\mathbb{R}^n$ with $A\mathbf{v}=\lambda\mathbf{v}$.

The $\lambda$-eigenspace is $\text{Nul}(A-\lambda I)$. Always a subspace of $\mathbb{R}^n$.

MEMORIZE

Characteristic Polynomial. $\det(A-xI)$. Eigenvalues are roots.

Algebraic multiplicity of $\lambda$: power of $(x-\lambda)$ in $\det(A-xI)$.

Geometric multiplicity of $\lambda$: $\dim(\text{Nul}(A-\lambda I))$. Always: geometric $\leq$ algebraic.

Proof Recipe — Eigenvalue proofs

Template: "Show $\lambda^k$ is eigenvalue of $A^k$"
1(2 pts) "By definition, $\exists$ nonzero $\mathbf{v}$ s.t. $A\mathbf{v}=\lambda\mathbf{v}$." — Don't forget "nonzero"!
2(2 pts) Multiply both sides by $A$ repeatedly: $A^k\mathbf{v}=\lambda^k\mathbf{v}$.
3(1 pt) "Since $\mathbf{v}\neq\mathbf{0}$, $\mathbf{v}$ is an eigenvector of $A^k$ with eigenvalue $\lambda^k$." $\square$

True / False Practice

10. $A$ is invertible if and only if $0$ is not an eigenvalue of $A$. (Quiz 3)
True (Quiz 3)

$A$ invertible $\Leftrightarrow$ no nonzero solution to $A\mathbf{x}=0\cdot\mathbf{x}$ $\Leftrightarrow$ $0$ not an eigenvalue.

11. If $-1$ is an eigenvalue of $A$, then $A+I$ is not invertible. (Quiz 4)
True (Quiz 4)

$(A+I)\mathbf{v}=A\mathbf{v}+\mathbf{v}=-\mathbf{v}+\mathbf{v}=\mathbf{0}$. So $A+I$ has nontrivial null space.

5.4

Diagonalization

HW 5 §5.4 · Midterm Q8
MEMORIZE

Diagonalizable. $A\in M_{n\times n}$ is diagonalizable if $A=PDP^{-1}$ for some invertible $P$ and diagonal $D$. Columns of $P$ are eigenvectors; diagonal entries of $D$ are eigenvalues.

Theorem 7 (Mezo's key theorem)

$A$ is diagonalizable $\Leftrightarrow$ for each eigenvalue $\lambda_i$: geometric multiplicity $=$ algebraic multiplicity.

Sufficient: $n$ distinct eigenvalues $\Rightarrow$ always diagonalizable.

Not diagonalizable over $\mathbb{R}$: if characteristic polynomial has irreducible quadratic factors — cite Theorem 7(b).

Worked Example — Midterm Q8

Midterm Q8 — exact6 marks

$A=\begin{pmatrix}4&0&0\\0&0&-1\\0&1&0\end{pmatrix}$. (a) Find characteristic polynomial and eigenvalues. (b) Is $A$ diagonalizable?

Solution
a$\det(A-xI)=(4-x)(x^2+1)$. Real eigenvalue: $\lambda=4$ only.
b$x^2+1$ is irreducible over $\mathbb{R}$. By Theorem 7(b), $A$ is not diagonalizable over $\mathbb{R}$.
5.5

Complex Numbers & Complex Eigenvalues

HW 5 §5.5 · Quiz 5
Complex Number Basics

$z=a+bi$, $\overline{z}=a-bi$, $|z|^2=a^2+b^2=z\overline{z}$. Polar form: $z=re^{i\theta}$. De Moivre: $z^k=r^ke^{ik\theta}$.

Conjugate Eigenvalue Theorem

If $A$ has real entries and $\lambda\in\mathbb{C}$ is an eigenvalue with eigenvector $\mathbf{v}\in\mathbb{C}^n$, then $\overline{\lambda}$ is also an eigenvalue with eigenvector $\overline{\mathbf{v}}$. Complex eigenvalues of real matrices always come in conjugate pairs.

12. Only linear transformations on finite-dimensional vector spaces can have eigenvectors. (Quiz 5)
False (Quiz 5)

$D(e^{\lambda x})=\lambda e^{\lambda x}$ — the derivative operator on infinite-dimensional function spaces has eigenvectors.

6.1

Inner Products, Norms & Distance

HW 6 · Quiz 6
MEMORIZE — 4 Axioms

$\langle\cdot,\cdot\rangle:V\times V\to\mathbb{R}$ (or $\mathbb{C}$) is an inner product if:

(1) $\langle\mathbf{u},\mathbf{v}\rangle=\overline{\langle\mathbf{v},\mathbf{u}\rangle}$   (2) $\langle\mathbf{u}+\mathbf{v},\mathbf{w}\rangle=\langle\mathbf{u},\mathbf{w}\rangle+\langle\mathbf{v},\mathbf{w}\rangle$

(3) $\langle c\mathbf{u},\mathbf{v}\rangle=c\langle\mathbf{u},\mathbf{v}\rangle$   (4) $\langle\mathbf{v},\mathbf{v}\rangle\geq0$ and $=0\Leftrightarrow\mathbf{v}=\mathbf{0}$

Norm: $\|\mathbf{v}\|=\sqrt{\langle\mathbf{v},\mathbf{v}\rangle}$  ·  Distance: $\text{dist}(\mathbf{u},\mathbf{v})=\|\mathbf{u}-\mathbf{v}\|$

⚠ Complex inner product — Mezo's convention

Standard inner product on $\mathbb{C}^n$: $\langle\mathbf{u},\mathbf{v}\rangle=u_1\overline{v_1}+\cdots+u_n\overline{v_n}$. Conjugate goes on the second slot.

Key Inequalities

Cauchy-Schwarz: $|\langle\mathbf{u},\mathbf{v}\rangle|\leq\|\mathbf{u}\|\|\mathbf{v}\|$  ·  Pythagorean theorem: If $\langle\mathbf{u},\mathbf{v}\rangle=0$ then $\|\mathbf{u}+\mathbf{v}\|^2=\|\mathbf{u}\|^2+\|\mathbf{v}\|^2$

True / False Practice

13. If $\|\mathbf{u}\|^2+\|\mathbf{v}\|^2=\|\mathbf{u}+\mathbf{v}\|^2$, then $\mathbf{u}\perp\mathbf{v}$. (Quiz 6)
True

Expand: $\|\mathbf{u}+\mathbf{v}\|^2=\|\mathbf{u}\|^2+2\langle\mathbf{u},\mathbf{v}\rangle+\|\mathbf{v}\|^2$. Equal means $2\langle\mathbf{u},\mathbf{v}\rangle=0$.

14. Two vectors orthogonal in the standard inner product on $\mathbb{R}^2$ are orthogonal in every inner product. (Quiz 9)
False (Quiz 9)

$(1,1)$ and $(1,-1)$: standard product gives 0. Under $\langle\mathbf{u},\mathbf{v}\rangle=4u_1v_1+5u_2v_2$: value is $4-5=-1\neq0$.

6.2

Orthogonal Sets & Orthogonal Complements

HW 7 · Quiz 7
MEMORIZE

Orthogonal set: $\langle\mathbf{u}_i,\mathbf{u}_j\rangle=0$ for all $i\neq j$.   Orthonormal: orthogonal AND $\|\mathbf{u}_i\|=1$ for all $i$.

Orthogonal complement: $W^\perp=\{\mathbf{v}:\langle\mathbf{v},\mathbf{w}\rangle=0\text{ for all }\mathbf{w}\in W\}$.

Key Theorem

Any orthogonal set of nonzero vectors is linearly independent.

If $\mathcal{B}=\{\mathbf{u}_1,\ldots,\mathbf{u}_k\}$ is orthogonal and $\mathbf{y}\in W=\text{Span}(\mathcal{B})$: $\mathbf{y}=\sum\frac{\langle\mathbf{y},\mathbf{u}_j\rangle}{\|\mathbf{u}_j\|^2}\mathbf{u}_j$. If orthonormal: denominators disappear.

15. Normalizing an orthogonal set may destroy orthogonality. (Quiz 7)
False (Quiz 7)

$\langle\frac{\mathbf{u}}{\|\mathbf{u}\|},\frac{\mathbf{v}}{\|\mathbf{v}\|}\rangle=\frac{1}{\|\mathbf{u}\|\|\mathbf{v}\|}\langle\mathbf{u},\mathbf{v}\rangle=0$. Scaling never creates non-zero inner products.

6.3

Orthogonal Projections & Best Approximation

HW 8 · Quiz 8
MEMORIZE

Projection onto $W$. If $\{\mathbf{u}_1,\ldots,\mathbf{u}_k\}$ is an orthogonal basis for $W$:

$$\text{proj}_W\mathbf{y}=\sum_{j=1}^k\frac{\langle\mathbf{y},\mathbf{u}_j\rangle}{\|\mathbf{u}_j\|^2}\mathbf{u}_j$$

Orthogonal decomposition: $\mathbf{y}=\text{proj}_W\mathbf{y}+\mathbf{z}$ where $\mathbf{z}=\mathbf{y}-\text{proj}_W\mathbf{y}\in W^\perp$.

Best Approximation Theorem

The closest point in $W$ to $\mathbf{y}$ is $\text{proj}_W\mathbf{y}$. The distance from $\mathbf{y}$ to $W$ is $\|\mathbf{y}-\text{proj}_W\mathbf{y}\|$.

16. For any $\mathbf{y}$ and subspace $W$, the vector $\mathbf{y}-\text{proj}_W\mathbf{y}$ is orthogonal to $W$. (Quiz 8 Q1)
True (Quiz 8)

By definition of the orthogonal projection, $\mathbf{y}-\text{proj}_W\mathbf{y}\in W^\perp$.

6.4

Gram-Schmidt Process

HW 8 §6.3–6.4 · Notes Mar 10–12
MEMORIZE — Algorithm

Given a basis $\{\mathbf{x}_1,\ldots,\mathbf{x}_k\}$ for $W$, produce orthogonal basis $\{\mathbf{v}_1,\ldots,\mathbf{v}_k\}$:

$$\mathbf{v}_1=\mathbf{x}_1 \qquad \mathbf{v}_j=\mathbf{x}_j-\sum_{i=1}^{j-1}\frac{\langle\mathbf{x}_j,\mathbf{v}_i\rangle}{\|\mathbf{v}_i\|^2}\mathbf{v}_i$$

To get orthonormal: $\mathbf{w}_j=\mathbf{v}_j/\|\mathbf{v}_j\|$. Always verify $\langle\mathbf{v}_i,\mathbf{v}_j\rangle=0$ after each step.

Inner Product Spaces — Function Spaces

⚠ Useful Integral Facts

Worked Example — Quiz 9

Projection in $C[-1,1]$Quiz 9

$\langle f,g\rangle=\int_{-1}^1 f(t)g(t)\,dt$. $p_0=1$, $p_1=x$, $p_2=x^2$. Find $\text{proj}_W p_2$ where $W=\text{Span}\{p_0,p_1\}$.

3$\text{proj}_W p_2=\frac{2/3}{2}\cdot 1+0\cdot x=\frac{1}{3}$. Best approximation to $x^2$ by degree $\leq1$ is the constant $\frac{1}{3}$.
7.1

Symmetric & Orthogonal Matrices

HW 10 · Notes Mar 19
MEMORIZE

Symmetric: $A=A^\top$.   Orthogonal matrix: $P^\top P=I$, equivalently $P^{-1}=P^\top$. Columns form an orthonormal basis for $\mathbb{R}^n$.

Properties of Orthogonal Matrices

$P$ orthogonal $\Rightarrow$ $\langle P\mathbf{u},P\mathbf{v}\rangle=\langle\mathbf{u},\mathbf{v}\rangle$ and $\|P\mathbf{u}\|=\|\mathbf{u}\|$.

$\det(P)=\pm1$. Rotation matrix: $P=\begin{pmatrix}\cos\theta&-\sin\theta\\\sin\theta&\cos\theta\end{pmatrix}$.

True / False Practice

17. If $P$ is orthogonal, then $P\mathbf{u}\perp P\mathbf{v}$ whenever $\mathbf{u}\perp\mathbf{v}$.
True

$\langle P\mathbf{u},P\mathbf{v}\rangle=\mathbf{v}^\top P^\top P\mathbf{u}=\mathbf{v}^\top\mathbf{u}=\langle\mathbf{u},\mathbf{v}\rangle=0$.

18. If $\det(P)=1$ then $P$ is orthogonal.
False

$\det=1$ is necessary but not sufficient. $\begin{pmatrix}2&0\\0&1/2\end{pmatrix}$ has $\det=1$ but $P^\top P=\begin{pmatrix}4&0\\0&1/4\end{pmatrix}\neq I$.

7.2

Spectral Theorem & Orthogonal Diagonalization

HW 11 · Notes Mar 24
MEMORIZE — Spectral Theorem

$A\in M_{n\times n}$ is orthogonally diagonalizable ($A=PDP^\top$, $P$ orthogonal) $\Leftrightarrow$ $A$ is symmetric ($A=A^\top$).

Theorem 1 — Key Lemma

If $A=A^\top$ and $\lambda_1\neq\lambda_2$ are distinct eigenvalues with eigenvectors $\mathbf{v}_1,\mathbf{v}_2$, then $\langle\mathbf{v}_1,\mathbf{v}_2\rangle=0$.

Proof key move: $\lambda_1\langle\mathbf{v}_1,\mathbf{v}_2\rangle=\langle A\mathbf{v}_1,\mathbf{v}_2\rangle\underbrace{=}_{A=A^\top}\langle\mathbf{v}_1,A\mathbf{v}_2\rangle=\lambda_2\langle\mathbf{v}_1,\mathbf{v}_2\rangle$. Since $\lambda_1\neq\lambda_2$: inner product $=0$.

Full Orthogonal Diagonalization Recipe

End-to-End: Orthogonally Diagonalize a Symmetric Matrix
1Compute $\det(A-xI)$. All roots are real (Spectral Theorem guarantees this).
2For each eigenvalue, find a basis for $\text{Nul}(A-\lambda I)$.
3If eigenspace has $\dim>1$, apply Gram-Schmidt within it. (Different eigenspaces are automatically orthogonal by Theorem 1.)
4Normalize all eigenvectors. Put them as columns of $P$.
5$D=$ diagonal matrix of corresponding eigenvalues. Then $A=PDP^\top$.

True / False Practice

19. Every real symmetric matrix is diagonalizable.
True — Spectral Theorem

Every real symmetric matrix is orthogonally diagonalizable — a stronger statement. All eigenvalues are real, and the Spectral Theorem guarantees an orthonormal basis of eigenvectors.

20. If $A$ is orthogonally diagonalizable, then $A$ is symmetric.
True — Biconditional

$A=PDP^\top$ with $P$ orthogonal $\Rightarrow$ $A^\top=PD^\top P^\top=PDP^\top=A$. The Spectral Theorem is an iff.

7.3

Quadratic Forms

Notes Mar 26 · NOT in textbook section
🔴 NEW — From Mar 26 Lecture

This topic was not in the original study guide. It comes directly from the Mar 26 notes and is almost certainly on the final.

MEMORIZE — Definition

Quadratic Form. Let $A\in M_{n\times n}$ be symmetric ($A=A^\top$). The quadratic form corresponding to $A$ is the function $Q:\mathbb{R}^n\to\mathbb{R}$ defined by:

$$Q(\mathbf{v}) = \mathbf{v}^\top A\mathbf{v}$$

$Q$ is a homogeneous polynomial of degree 2 in the entries of $\mathbf{v}$.

Reading off the matrix from a polynomial

For $A=\begin{pmatrix}a&b\\b&c\end{pmatrix}$ (symmetric), the quadratic form is:

$$Q(\mathbf{v}) = ax_1^2 + 2bx_1x_2 + cx_2^2$$

Rule: diagonal entries of $A$ → coefficients of squared terms. Off-diagonal entries $a_{jk}$ → half the coefficient of the mixed term $x_jx_k$ (factor of 2 because both $(j,k)$ and $(k,j)$ entries contribute).

💡 Converting polynomial → matrix

Given a homogeneous degree-2 polynomial: put the coefficient of $x_i^2$ on the diagonal. Split each mixed monomial coefficient in half: coefficient of $x_ix_j$ gives $\frac{1}{2}\cdot\text{coeff}$ in positions $(i,j)$ and $(j,i)$.

Principal Axes Theorem

MEMORIZE — Theorem 4

Principal Axes Theorem. Let $A\in M_{n\times n}$ be symmetric with orthogonal diagonalization $A=PDP^\top$ (columns of $P$ are orthonormal eigenvectors $\mathbf{u}_1,\ldots,\mathbf{u}_n$, $D=\text{diag}(\lambda_1,\ldots,\lambda_n)$). Then:

$$Q(\mathbf{v}) = \mathbf{v}^\top A\mathbf{v} = \mathbf{y}^\top D\mathbf{y} = \lambda_1 y_1^2 + \lambda_2 y_2^2 + \cdots + \lambda_n y_n^2$$

where $\mathbf{y} = P^\top\mathbf{v}$ (equivalently $\mathbf{v} = P\mathbf{y}$, i.e. $\mathbf{y}=[\mathbf{v}]_\mathcal{B}$ in the eigenvector basis $\mathcal{B}$).

Geometric meaning: After changing to the eigenvector coordinate system, the quadratic form has no cross terms — it's a pure sum of squares scaled by eigenvalues.

Spectral Decomposition Connection

From the spectral decomposition $A = \lambda_1\mathbf{u}_1\mathbf{u}_1^\top + \cdots + \lambda_n\mathbf{u}_n\mathbf{u}_n^\top$, we get:

$$Q(\mathbf{v}) = \lambda_1(\mathbf{u}_1\cdot\mathbf{v})^2 + \cdots + \lambda_n(\mathbf{u}_n\cdot\mathbf{v})^2 = \lambda_1\|\text{proj}_{\mathbf{u}_1}\mathbf{v}\|^2 + \cdots$$

Worked Example — Principal Axes

Solution

4In principal axes coordinates: $Q=7y_1^2+3y_2^2$. No cross term! The axes of symmetry are $\text{span}\{\mathbf{u}_1\}$ and $\text{span}\{\mathbf{u}_2\}$.

7.4

Positive Definite, Negative Definite, Indefinite

Notes Mar 31 · Theorem 5
🔴 NEW — From Mar 31 Lecture

Classifying quadratic forms by their sign. Determined entirely by the eigenvalues of $A$.

MEMORIZE — Definitions

Let $Q(\mathbf{v})=\mathbf{v}^\top A\mathbf{v}$ be the quadratic form of symmetric $A$. Then:

• $Q$ is positive definite if $Q(\mathbf{v})>0$ for all $\mathbf{v}\neq\mathbf{0}$.

• $Q$ is negative definite if $Q(\mathbf{v})<0$ for all $\mathbf{v}\neq\mathbf{0}$.

• $Q$ is indefinite if $Q$ takes both positive and negative values.

(Also: positive semi-definite $Q\geq0$; negative semi-definite $Q\leq0$.)

MEMORIZE — Theorem 5 (Key Theorem)

Let $Q=\mathbf{v}^\top A\mathbf{v}$ for symmetric $A\in M_{n\times n}$. Then:

(a) $Q$ is positive definite $\Leftrightarrow$ all eigenvalues of $A$ are positive ($>0$).

(b) $Q$ is negative definite $\Leftrightarrow$ all eigenvalues of $A$ are negative ($<0$).

(c) $Q$ is indefinite $\Leftrightarrow$ $A$ has both positive and negative eigenvalues.

Why: By the Principal Axes Theorem, $Q=\lambda_1y_1^2+\cdots+\lambda_ny_n^2$. The sign of $Q$ is determined entirely by the signs of the $\lambda_i$.

⚠ Connection to Inner Products

A symmetric bilinear form $B(\mathbf{u},\mathbf{v})=\mathbf{u}^\top A\mathbf{v}$ is an inner product if and only if its quadratic form $Q(\mathbf{v})=B(\mathbf{v},\mathbf{v})=\mathbf{v}^\top A\mathbf{v}$ is positive definite — i.e., all eigenvalues of $A$ are positive.

Symmetric Bilinear Forms

🔴 NEW — Last Lecture (Apr 2) — Not in Textbook

MEMORIZE — Definition

Symmetric Bilinear Form. A function $B:V\times V\to\mathbb{R}$ is a symmetric bilinear form if:

(i) $B(\mathbf{u},\mathbf{v})=B(\mathbf{v},\mathbf{u})$  (symmetric)

(ii) $B(\mathbf{u}+\mathbf{v},\mathbf{w})=B(\mathbf{u},\mathbf{w})+B(\mathbf{v},\mathbf{w})$  (additive in first slot)

(iii) $B(c\mathbf{u},\mathbf{v})=cB(\mathbf{u},\mathbf{v})$ for $c\in\mathbb{R}$  (scalar in first slot)

Key consequence: By symmetry + (ii)+(iii), $B$ is also linear in the second slot. That's why it's called bilinear.

⚠ Bilinear Form vs Inner Product

A symmetric bilinear form is NOT necessarily an inner product — it can fail condition (iv): $B(\mathbf{v},\mathbf{v})\geq0$ with $=0\Leftrightarrow\mathbf{v}=\mathbf{0}$.

Example from notes: $B(\mathbf{u},\mathbf{v})=\mathbf{u}^\top A\mathbf{v}$ for $A=\begin{pmatrix}0&1\\1&0\end{pmatrix}$. Check: $B((1,0)^\top,(1,0)^\top)=(1,0)\begin{pmatrix}0&1\\1&0\end{pmatrix}\begin{pmatrix}1\\0\end{pmatrix}=0$, but $(1,0)^\top\neq\mathbf{0}$. Property (iv) fails — so not an inner product.

MEMORIZE — Theorem (Matrix Representation)

Every symmetric bilinear form on $\mathbb{R}^n$ is of the form $B(\mathbf{u},\mathbf{v})=\mathbf{u}^\top A\mathbf{v}$ for a unique symmetric matrix $A\in M_{n\times n}$.

Finding $A$: The $(j,k)$-entry of $A$ is $a_{jk}=B(\mathbf{e}_j,\mathbf{e}_k)$ where $\mathbf{e}_1,\ldots,\mathbf{e}_n$ are standard basis vectors. Symmetry of $B$ forces $A=A^\top$.

Conversely: Any symmetric $A$ defines a symmetric bilinear form via $B(\mathbf{u},\mathbf{v})=\mathbf{u}^\top A\mathbf{v}$.

Connection: Bilinear Form ↔ Quadratic Form

Every symmetric bilinear form $B$ has an associated quadratic form:

$$Q(\mathbf{v}) = B(\mathbf{v},\mathbf{v}) = \mathbf{v}^\top A\mathbf{v}$$

You can recover $B$ from $Q$ via the polarization identity: $B(\mathbf{u},\mathbf{v})=\frac{1}{2}[Q(\mathbf{u}+\mathbf{v})-Q(\mathbf{u})-Q(\mathbf{v})]$.

When is a Bilinear Form an Inner Product?

$B(\mathbf{u},\mathbf{v})=\mathbf{u}^\top A\mathbf{v}$ is an inner product on $\mathbb{R}^n$ if and only if its quadratic form $Q(\mathbf{v})=\mathbf{v}^\top A\mathbf{v}$ is positive definite, i.e., all eigenvalues of $A$ are positive.

Worked Examples

Example 1 — Apr 2 NotesVerifying bilinear form

Let $A\in M_{n\times n}$ be any matrix. Define $B(\mathbf{u},\mathbf{v})=\mathbf{u}^\top A\mathbf{v}$. Is $B$ always a symmetric bilinear form?

Solution
1Bilinear: $B(\mathbf{u}+\mathbf{v},\mathbf{w})=(\mathbf{u}+\mathbf{v})^\top A\mathbf{w}=\mathbf{u}^\top A\mathbf{w}+\mathbf{v}^\top A\mathbf{w}=B(\mathbf{u},\mathbf{w})+B(\mathbf{v},\mathbf{w})$ ✓. Similarly for scalar and second slot.
2Symmetric? $B(\mathbf{u},\mathbf{v})=\mathbf{u}^\top A\mathbf{v}$. $B(\mathbf{v},\mathbf{u})=\mathbf{v}^\top A\mathbf{u}=(\mathbf{u}^\top A^\top\mathbf{v})^\top$. Since this is a $1\times1$ matrix: $=\mathbf{u}^\top A^\top\mathbf{v}$. So $B(\mathbf{u},\mathbf{v})=B(\mathbf{v},\mathbf{u})$ if and only if $A=A^\top$.
3Conclusion: $B$ is bilinear always, but symmetric only when $A$ is symmetric.
Example 2 — Apr 2 NotesIs it an inner product?

$A=\begin{pmatrix}1&3\\3&1\end{pmatrix}$. $B(\mathbf{u},\mathbf{v})=\mathbf{u}^\top A\mathbf{v}$. Is $B$ an inner product on $\mathbb{R}^2$?

Solution
1$A=A^\top$ ✓, so $B$ is a symmetric bilinear form.
2Eigenvalues: $\det(A-\lambda I)=(\lambda-1)^2-9=\lambda^2-2\lambda-8=(\lambda-4)(\lambda+2)$. Eigenvalues: $\lambda_1=4$ and $\lambda_2=-2$.
3Since $\lambda_2=-2<0$, the quadratic form $Q$ is indefinite (not positive definite). Therefore $B$ is not an inner product. Specifically, $Q(\mathbf{v}_2)<0$ for the eigenvector $\mathbf{v}_2$ of $\lambda_2=-2$.

True / False Practice

26. Every inner product on $\mathbb{R}^n$ is a symmetric bilinear form.
True

An inner product satisfies symmetry and bilinearity (conditions i–iii of a symmetric bilinear form) plus the extra positivity condition (iv). So every inner product is automatically a symmetric bilinear form — but not vice versa.

27. $B(\mathbf{u},\mathbf{v})=\mathbf{u}^\top A\mathbf{v}$ for symmetric $A$ is an inner product if and only if all eigenvalues of $A$ are positive.
True

$B$ is an inner product $\Leftrightarrow$ $Q(\mathbf{v})=\mathbf{v}^\top A\mathbf{v}>0$ for all $\mathbf{v}\neq\mathbf{0}$ $\Leftrightarrow$ $Q$ is positive definite $\Leftrightarrow$ all eigenvalues of $A>0$ (Theorem 5).

28. If $B$ is a symmetric bilinear form that is not an inner product, then there must exist $\mathbf{v}\neq\mathbf{0}$ with $B(\mathbf{v},\mathbf{v})\leq0$.
True

The only condition separating a symmetric bilinear form from an inner product is positive definiteness. If it fails, there exists $\mathbf{v}\neq\mathbf{0}$ with $Q(\mathbf{v})=B(\mathbf{v},\mathbf{v})\leq0$.

P-1 — Subspace proof
§4.1 midterm Q4

Let $V$ be a real vector space. Show that $W = \{f\in V : f(0)=f(1)=0\}$ is a subspace of $V$.

Proof
1$z(x)=0$: $z(0)=z(1)=0$. So $z\in W$.
2Let $f,g\in W$. $(f+g)(0)=0$, $(f+g)(1)=0$. So $f+g\in W$.
3Let $f\in W$, $c\in\mathbb{R}$. $(cf)(0)=cf(0)=0$, $(cf)(1)=0$. So $cf\in W$. $\square$
P-2 — ker(T) is a subspace
HW5 §4.2

Let $T:V\to W$ be linear. Show $\ker(T)$ is a subspace of $V$.

Proof
1$T(\mathbf{0})=0\cdot T(\mathbf{0})=\mathbf{0}$, so $\mathbf{0}\in\ker T$.
2Let $\mathbf{u},\mathbf{v}\in\ker T$. $T(\mathbf{u}+\mathbf{v})=T(\mathbf{u})+T(\mathbf{v})=\mathbf{0}$. So $\mathbf{u}+\mathbf{v}\in\ker T$.
3Let $\mathbf{u}\in\ker T$, $c\in\mathbb{R}$. $T(c\mathbf{u})=cT(\mathbf{u})=\mathbf{0}$. So $c\mathbf{u}\in\ker T$. $\square$

P-5 — Spectral Theorem key lemma
HW11 / Notes Mar 24

Let $A=A^\top$. Suppose $A\mathbf{v}_1=\lambda_1\mathbf{v}_1$, $A\mathbf{v}_2=\lambda_2\mathbf{v}_2$ with $\lambda_1\neq\lambda_2$. Show $\langle\mathbf{v}_1,\mathbf{v}_2\rangle=0$.

Proof
1$\lambda_1\langle\mathbf{v}_1,\mathbf{v}_2\rangle=\langle\lambda_1\mathbf{v}_1,\mathbf{v}_2\rangle=\langle A\mathbf{v}_1,\mathbf{v}_2\rangle$.
2Since $A=A^\top$: $\langle A\mathbf{v}_1,\mathbf{v}_2\rangle=(A\mathbf{v}_1)^\top\mathbf{v}_2=\mathbf{v}_1^\top A^\top\mathbf{v}_2=\langle\mathbf{v}_1,A\mathbf{v}_2\rangle=\lambda_2\langle\mathbf{v}_1,\mathbf{v}_2\rangle$.
3$(\lambda_1-\lambda_2)\langle\mathbf{v}_1,\mathbf{v}_2\rangle=0$. Since $\lambda_1\neq\lambda_2$: $\langle\mathbf{v}_1,\mathbf{v}_2\rangle=0$. $\square$
